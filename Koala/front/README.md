## Landing Jardín Infantil Koala

Landing page estática para Jardín Infantil y Sala Cuna Koala, pensada para correrse en local y mostrar:

- **Header** con navegación y llamado a la acción.
- **Sección principal** con foto del jardín y mensaje principal.
- **Sección sobre nosotros**, **servicios**, **galería**, **testimonios** y **contacto**, siguiendo el diseño de las imágenes adjuntas.

### Estructura

- `index.html`: layout principal y contenido de la landing.
- `styles.css`: estilos globales y diseño responsivo.
- `assets/`: imágenes usadas en la página (logo, header, cards y galería).

### Cómo ejecutarla en local

- **Opción 1 (rápida)**: abrir directamente `index.html` en el navegador.
- **Opción 2 (servidor estático)**:
  1. Asegúrate de tener Node.js instalado.
  2. En esta carpeta, ejecuta:

     ```bash
     npm run start
     ```

  3. Abre la URL que indique `serve` (por defecto suele ser `http://localhost:3000` o similar).
